# Darya Beigi | Software Engineer Portfolio

A modern, single-page developer portfolio built to showcase over six years of full-stack engineering experience, featuring professional work and personal projects.

Live Site: [dbeigi.com](https://dbeigi.com) 

## Tech Stack
This project was built from the ground up prioritizing speed, modern tooling, and a minimalist design:
* **Framework:** Vue 3
* **Build Tool:** Vite
* **Styling:** Tailwind CSS
* **Hosting:** Netlify
